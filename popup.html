ℹ️ GUC Grading & GPA Calculation

GPA is calculated using the lowest value for each letter grade:
A+= 0.7A= 1.0A-= 1.3
B+= 1.7B= 2.0B-= 2.3
C+= 2.7C= 3.0C-= 3.3
D= 3.7F= 5.0
For example, if you get an A+ (0.7–1.0), your GPA is calculated using 0.7.
